
# 📝 Python To-Do List CLI

A simple and beginner-friendly command-line To-Do List application built with Python.

## ✨ Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- ✏️ Edit tasks
- 🗑️ Delete tasks
- 🔍 Search tasks
- 📊 View task statistics
- 💾 Automatically saves tasks to a JSON file

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Functions
- Loops
- Error Handling

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/python-todo-list.git
```

2. Open the project folder

```bash
cd python-todo-list
```

3. Run the program

```bash
python todo.py
```
